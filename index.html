Dropdown examples

Dropdown with 100 users and data source. 

window.d1 = new dropdown.Dropdown({
    container : "dropdown-container-1",
    autoRender : true,
    data : data,
    showAvatars : true,
    dataSourceUrl : '/suggest'
});

Tiny dropdown with single select and 30 users. Data source also provided. 

window.d2 = new dropdown.Dropdown({

Dropdown with 100 users without data source with maxSize set to 5. 

window.d3 = new dropdown.Dropdown({
    container : "dropdown-container-3",
    autoRender : true,
    data : data3,
    showAvatars : true,
